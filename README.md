# Mynfo


## Mynfo Overview

マークダウン記法を使った CMS


## How to run on localhost

- `$ git clone https://github.com/dotnsf/mynfo.git`

- `$ cd mynfo`

- `$ npm install`

- (Edit contents under `md/` folder)

- (Add/Edit contents under `public/` folder, if needed.)

- `$ node app`

- `http://localhost:8080/`
